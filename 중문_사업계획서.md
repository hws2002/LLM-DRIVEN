# 项目名称

GraphNode: AI**驱动的知识网络可视化平台**

# 项目介绍

GraphNode是一种“第二大脑”（Second Brain）系统，能够自动分析用户与生成式人工智能（ChatGPT）的对话记录以及个人笔记，并将其转化为以思维导图为结构的知识图谱。

不同于传统的信息存储工具，GraphNode不仅对信息进行整理与保存，更通过可视化方式呈现知识单元之间的关联关系，从而实现信息的资产化，并支持用户进行直观且高效的探索与发现。

# 项目背景

在信息密集型工作场景中，用户平均每天需花费约3.6小时（约占工作时间的45%）用于搜索所需信息（Coveo，2024）。此外，在数字信息过载的背景下，近90%的已获取信息会在一周内被遗忘（eLearning Industry，2024）。

以 ChatGPT 为代表的生成式人工智能工具正在被迅速普及，带来了数据产出的指数级增长。然而，由于缺乏有效的工具对这些信息进行系统化整理与关联，大量信息仍停留在碎片化、短暂性的消费状态之中，难以沉淀为长期可复用的知识资产。

针对这一问题，GraphNode提出了一种基于人工智能的“第二大脑”解决方案，能够自动将分散的AI对话记录与个人笔记转化为可视化的知识图谱，从而将转瞬即逝的信息转化为可持续积累、可导航的知识资产。

# 项目核心功能与服务

当前个人知识管理（PKM）市场呈现出明显的两极分化格局，而两类主流解决方案均存在关键性局限。

以 ChatGPT 为代表的生成式人工智能工具在信息生成方面表现卓越，但无法跨对话构建知识之间的关联关系，导致用户获得的信息呈现出碎片化、短暂性的特征。

以 Obsidian、Logseq 和 Roam Research 为代表的知识管理工具虽然提供了图谱与链接功能，但高度依赖用户手动构建所有知识连接，使用门槛较高。

截至2025年，Obsidian 年增长率已放缓至约22%，Roam Research 仍局限于少数重度用户群体，而 Logseq 在2023年数据库转型后出现核心用户流失。由此可以看出，现有 PKM 工具难以突破小众高强度用户圈层，其根本原因在于“手动结构化”这一使用门槛限制了规模化增长。

在这一背景下，GraphNode 位于生成式 AI 与知识管理工具的交汇点，通过人工智能自动构建语义关联，实现两类工具单独无法完成的能力整合。

GraphNode的核心功能：

**1. 自动化知识图谱生成**

基于自研的八阶段 AI 流水线，系统可从语义层面对用户的对话记录与笔记进行分析，自动完成概念提取、依赖关系识别、信息聚类，并构建具有实际意义的节点连接关系。用户只需自由输入内容，结构将自动生成。

**2. 跨平台数据整合**

GraphNode 作为元层平台，可打通 ChatGPT、Notion、Obsidian 等多平台数据，将分散的信息统一整合为一个连贯的知识图谱，从根本上解决多平台导致的语境割裂问题

**3. 交互式二维与三维可视化**

用户可通过直观的二维聚类视图快速浏览知识结构，或在三维空间中探索复杂关系网络。独特的细粒度知识图谱功能支持对单一节点进行深度解析，将长文本或研究内容拆解为由关键概念及其关系构成的子图结构。

**4. AI驱动的知识洞察**

不仅限于可视化展示，GraphNode还能对累积的知识进行分析，识别潜在模式、发现知识盲区，并提供个性化学习建议，使信息存储从“被动记录”转变为“主动智能”

GraphNode的技术差异化：

- **自研AI流水线**
    
    实现从概念提取、语义聚类、关系构建到个性化洞察生成的全流程自动化。目前相关知识产权（专利及著作权）正在申请中。
    
- **多语言知识映射能力**
    
    针对多语言语义理解进行优化，支持跨语言知识关联构建，为全球化市场拓展提供技术基础。
    
- **分层式知识结构设计**
    
    采用三层结构体系（Workspace → Cluster → Node），能够适配从个人笔记到组织级知识库的不同规模，实现高效的信息管理。
    

# 项目的商业化潜力

GraphNode 面向广泛的信息密集型用户群体。

其初期核心目标用户为韩国与中国高校的本科生及研究生。作为以学习与研究为主要任务的群体，他们需要自主处理大量信息，并在信息不断积累的过程中反复在碎片化语境中进行检索与重构。尤其是频繁使用生成式人工智能工具的用户，对知识结构化与系统化管理存在天然且迫切的需求。

调研结果进一步验证了这一需求：

- 在对清华大学与北京大学韩国留学生的问卷调查中，**69.2% 的受访者将“难以找回过往笔记的上下文”视为现有工具的最大痛点**
- **100% 的受访者将“学习与研究整理”作为核心使用场景**
- **92.3% 的受访者将“知识可视化”视为最期待的功能**

长期角度，GraphNode 将逐步拓展至专业用户群体，包括产品经理、开发者及设计师等。这类用户在工作中需要同时处理多类型信息，但在实际流程中，信息往往分散于多个平台，导致语境割裂、知识流转难以追踪，从而显著降低效率。

在市场进入策略上，GraphNode 采用Bottom-up的增长路径：

- 以学生群体的自然扩散作为初始增长动力
- 通过知识积累带来的复利效应提升用户粘性
- 实现从个人使用 → 团队协作 → 组织级应用的逐步扩展

**韩国市场进入策略:**

1. 通过大学社区平台（Everytime、Instagram）实现初期切入
2. 向学生用户群体扩散
3. 与全球AI平台（如 ChatGPT、Gemini）进行整合

依托团队在韩国高校既有的社群与人脉资源，GraphNode 将以首尔重点高校为核心切入点，优先覆盖高频使用AI工具的学生与研究生群体，构建校园级增长基础。

**中国市场进入策略:**

1. 通过清华大学、北京大学等高校的韩国留学生社群实现初期切入
2. 向本地中国学生用户扩展
3. 与中国本土AI平台（Qwen、Kimi）进行集成

借助团队在中国高校的在地网络资源，GraphNode将首先在顶尖高校建立初始用户基础，并以京津冀区域成熟的学术与人工智能产业生态为战略支点，逐步向全国市场扩展。同时，通过与Qwen、Kimi 等本土AI平台的API集成，实现产品本地化与SaaS系统的稳定对接，以适应中国企业级应用需求的快速增长趋势。

# BM

GraphNode 面向全球个人知识管理（PKM）软件市场。该市场规模约为**152亿美元**，年复合增长率约为**12%**。其中，亚太地区约占全球市场的**30%**，是本项目的重点切入区域。

GraphNode 采用分层订阅制，涵盖 **Free、Standard、Pro** 三个主要层级，并结合长期扩展模型，以满足不同用户需求与使用深度。

- Free

用于降低使用门槛并促进用户增长。该版本主要依赖本地存储，提供基础图谱生成能力，并限制最多 **50条笔记**。其核心目标在于引导用户快速上手并体验产品核心价值。

- Standard

定价约为**7美元/月**（年付享折扣）。提供无限笔记创建、云端同步以及每月AI积分额度。在保证一体化使用体验的同时，通过内部使用限制控制成本。

- Pro

定价约为**14美元/月** 或基于额外积分付费。面向高级用户及专业人群，支持复杂的 Agent 推理能力（研究分析与大规模数据处理）。针对高算力消耗任务，采用额外计费或更高阶使用层级，以避免收入流失。

从长期来看，GraphNode 将进一步拓展平台能力：

- 推出团队协作功能（图谱共享与实时协作）
- 构建平台生态（支持 Agent工作流）
- 建立知识图谱模板市场，形成 **基于交易抽成的 C2C 商业模式**
- 推出企业级授权，实现 **B2B 收入增长**

付费转化逻辑：

免费版本中的限制（如50条笔记上限、缺乏云同步功能）将自然引导用户升级至付费版本。同时，一年期学生免费许可的到期将成为早期关键变现路径，有效推动用户向付费订阅转化。

收入预测：

GraphNode当前处于**尚未产生收入阶段**，计划于**2026年4月MVP上线后启动商业化**。

- **第1年（2026）**：通过高校免费许可获取用户，目标付费转化率 **5%**
- **第2年（2027）**：韩国与中国市场达到 **3万付费用户**，实现 **300万美元 ARR**
- **第3年（2028）**：达到 **12万付费用户**，目标 **1440万美元 ARR**
